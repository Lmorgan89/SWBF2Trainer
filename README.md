# SWBF2Trainer
A game trainer for Star Wars Battlefront II Classic (2005)

This code can be compiled to a DLL which then can be injected into the game.
The current options available are:
- Inifinite Health
- Infinite Ammo
- Infinite Stamina
- Buggy Line ESP
